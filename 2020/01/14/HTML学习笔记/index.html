HTML学习笔记

发表于 2020-01-14 更新于 2020-02-03 分类于 WEB前端 阅读次数：

HTML学习笔记整理，包括链接、图像、表单、音频等，附上部分代码样例

编写链接

  • 点击a元素之间的内容可以链接到指定页面，链接用href
    • 指向其他网站，需要用绝对URL
    • 指向同一网站的其他页面，可以用相对URL
    • 连接到当前页面某个特定位置

添加图像

  • 使用<img>元素
    • src的特性值是图片的相对/绝对URL
    • alt对图像文本进行说明，在图像无法查看时会对图像进行描述
    • title光标停在图像上时会以提示的方式显示图像特性内容
  • HTML5引入<figure>元素用来包含图像以及对图像的说明，图像说明用<figcaption>元素

表单

说起表单，知名的就比如 Google 主页的搜索框与 百度 主页的搜索框了

  • 表单控件位于<form>元素中
    • action特性，特性是服务器页面的URL，用来接收提交的信息
    • method特性
      • get: 适用于段表单，只从服务器检索数据的情形
      • post: 上传文件，信息很长，有敏感信息（密码等），向数据库添加或删除信息
  • <input>元素
    • type特性
      • type="txt" 文本框
      • type="password" 密码框
      • type="textarea" 文本域
        • cols 文本域横向宽度
        • rows 文本域纵向行数
    • name特性
      • 表明密码框的名称，它将于用户输入的密码一同发送到服务器哦
    • size
    • maxlength
    • placeholder特性的值为文本输入控件上输入文本之前的值
  • <input>元素
    • 文件上传域type="file"
    • 提交按钮type="submit"
      • name特性（不是必须的）
      • value特性控制在按钮上显示的文本
    • 单选按钮type="radio"
    • 复选框type="checkbox"
      • name特性与用户选择项的值一并发送到服务器
      • value特性指定被选中选项要发送到服务器的值
      • checked特性指定页面加载时被选中的值
    • 图像按钮type="image"，其余用法和<img>元素一样
    • 隐藏表单控件type="hidden"
    • 日期控件type="date"
    • 电子邮件控件type="email"
    • URL输入控件type="url"
    • 搜索输入控件type="search"
  • 下拉列表框<select>
    • name特性：表单控件名称，与选择值一并发送到服务器
    • multiple特性允许多选
    • <option>元素指定用户可以选择的选项
      • value
      • selected
      • size
  • 按钮<botton>元素
    • 可以更好控制按钮显示方式
    • 并允许其他元素出现在<button>元素内
  • 标签表单控件<label>
    • for特性声明标签控件标注的是哪个表单控件
  • 组合表单元素<fieldset>
    • <legend>元素包含一个标题

其他标记

  • 块级元素<div>
  • 内联元素<span>
  • 内联框架<iframe></iframe>
    • 就像一个小窗口，可以在其中看到另一个网页
    • src="https://xxx.xxx"
    • seamless特性：应用在不希望出现滚动条的地方
  • 页面信息<meta>
    • 位于<head>元素中，包含页面相关信息
    • name特性：要设定的属性，常用值：
      • description：包含页面描述信息
      • keywords：关键词列表，逗号分隔
      • robots：指定搜索引擎是否可以将页面加入搜索结果
        • noindex：不希望加入
        • nofollow：加入，但不要收录页面上链接的其他页面
    • content特性
    • http-equiv特性
      • author：网页设计者
      • pragma：防止浏览器对页面缓存
      • expires：指定页面过期时间以及缓存有效期
