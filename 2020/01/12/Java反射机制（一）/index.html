Java反射机制学习笔记（一），包括反射定义及优缺点、相关API介绍、利用反射打印一个类的全部信息等

什么是反射？

  • 在程序运行时，对任意一个实体类，都能获取该类的所有属性和方法；对任意一个对象，都能调用它的任意方法和属性。这种动态机制称为反射 (reflective)
  • 下面是 Oracle 官方文档解释：
    • Reflection enables Java code to discover information about the fields, methods and constructors of loaded classes, and to use reflected fields, methods, and constructors to operate on their underlying counterparts, within security restrictions.
    • The API accommodates applications that need access to either the public members of a target object (based on its runtime class) or the members declared by a given class. It also allows programs to suppress default reflective access control.
  • 列举反射的部分用途
    • JDBC中，利用反射动态加载了数据库驱动程序
    • Web服务器中利用反射调用了Sevlet的服务方法
    • IDEA等开发工具利用反射动态刨析对象的类型与结构，动态提示对象的属性和方法
    • 许多框架都用到反射机制，注入属性，调用方法

优缺点

  • 优点：
    • 反射属于动态编译，灵活性高
  • 缺点
    • 反射的操作主要通过 JVM 执行，时间成本会高于直接执行 java 代码

反射 API

java.lang.Class

  • 程序运行期间，系统始终为所有的对象维护一个被称为运行时的类型标识，这个信息跟踪着每个对象所属的类，虚拟机利用运行时类型信息选择相应方法执行。Class类就用来保存这些信息
  • Class 被 final 修饰，不可变
  • 其构造方法被 private 修饰， 仅由 Java 虚拟机创建Class对象
  • 介绍其部分方法
    • public Constructor<?>[] getDeclaredConstructors() throws SecurityException
    • public Method[] getDeclaredMethods() throws SecurityException
    • public Field[] getDeclaredFields() throws SecurityException
    • public String getName()
    • public Class<?> getType()
    • public int getModifiers()
      • 返回此类或接口的Java语言修饰符，以整数编码。
      • 常与 java.lang.reflect.Modifier 中的 toString() 连用

java.lang.reflect.Constructor

  • 源码可以看出其也是 final 型的
  • 部分常用方法
    • public T newInstance(Object ... initargs)
    • public Class<T> getDeclaringClass()
    • public String getName()
    • public int getModifiers()
    • public Class<?>[] getParameterTypes()
      • 返回一个用于描述返回类型的 Class 对象

java.lang.reflect.Method

  • 同样是 final 型的
  • 部分常用方法
    • public Class<?> getDeclaringClass()
    • public Class<?> getReturnType()
    • public int getModifiers()
    • public String getName()
    • public Class<?>[] getParameterTypes()

java.lang.reflect.Field

  • 同样是 final 型的
  • 部分常用方法
    • public Class<T> getDeclaringClass()
    • public String getName()
    • public int getModifiers()
    • public Class<?> getType()

java.lang.reflect.Modifier

  • The Modifier class provides static methods and constants to decode class and member access modifiers. The sets of modifiers are represented as integers with distinct bit positions representing different modifiers.
  • 介绍其几个常用方法
    • public static String toString​(int mod)
      • Return a string describing the access modifier flags in the specified modifier
