Java--lambda表达式方面笔记整理，包括了定义、几个常用函数式接口、方法引用等，附上一些示例

什么是lambda表达式

  • lambda表达式是一个可传递的代码块，可以在以后执行一次或多次，也可以叫做闭包

    一般语法

  • (parameters) -> { statements }
  • 如果表达式没有参数，仍然要提供空括号
    • () -> { … }

延迟执行deferred execution

  • 使用 lambda 表达式的重点是延迟执行（deferred execution）
    • 在一个单独的线程中运行代码
    • 多次运行代码
    • 在算法的适当位置运行代码
    • 发生某种情况时执行代码
    • 只要必要时才运行代码

函数式接口

什么是函数式接口

  • 对于只有一个抽象方法的接口称为函数式接口
  • 可以有其他静态方法或默认方法
  • java.util.function包中定义了很多非常通用的函数式接口
    • 可以发现函数式接口都有注解@FunctionalInterface，如：

java.util.function.Supplier

  • T get();

java.util.function.Consumer

  • void accept(T var1);
  • default Consumer<T> andThen(Consumer<? super T> after)
    • 注意：谁在前先消费谁
  • 实例：操作字符串，分别输出大写和小写

java.util.function.Function<T, R>

  • R apply(T var1);
    • 例子：字符串转为整型数
  • default <V> Function<V, R> compose(Function<? super V, ? extends T> before)
    • 例子：求出其方根的十六进制字符串表示形式
  • static <T> Function<T, T> identity()

java.util.function.Predicate

  • boolean test(T var1);
    • 例子：测试字符串长度
  • default Predicate<T> and(Predicate<? super T> other)
    • 例子：
  • default Predicate<T> negate()
  • default Predicate<T> or(Predicate<? super T> other)
  • static <T> Predicate<T> isEqual(Object targetRef)
  • static <T> Predicate<T> not(Predicate<? super T> target)

简要记录了四个常用函数式接口，当然还有很多，如：

  • java.util.function.BiConsumer
  • java.util.function.BiPredicate
  • java.util.function.UnaryOperator
  • java.util.function.BinaryOperator
  • 等等等等……

方法引用

类型 语法 对应的 lambda 表达式
静态方法引用 ClassName::staticMethod (args) -> {staticMethod(args)}
实例方法引用 ClassName::instanceMethod (args) -> {instance.instanceMethod(args)}
对象方法引用 ClassName::instanceMethod (instance, instanceMethod) -> ClassName.instanceMehod(args)
super引用父类方法 super::instanceMethod (args) -> super.instanceMethod(args)
this引用本类方法 this::instanceMethod (args) -> this.instanceMethod(args)
构造方法引用 ClassName::new (args) -> new ClassName(args)
数组的构造器引用 int[]::new len -> new int[len]
