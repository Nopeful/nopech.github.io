Hexo+Github搭建博客（基于Windows）

发表于 2020-01-09 更新于 2020-02-03 分类于 Hexo+Github博客 阅读次数：
本文字数： 1.4k 阅读时长 ≈ 1 分钟
Hexo+GitHub搭建博客教程，关于GitHub仓库，Git、nodejs、npm安装与使用等

一、GitHub账号和仓库

  • 到github官网注册账号
  • 新建仓库
    • Repository name: yourname.github.io

二、Git安装与使用（方便管理代码等）

  • 到 git官网下载
    • 选择 bit Git for Windows Setup.
  • 右击桌面，点击 Git Bash Here
  • 接着输入两条指令

三、配置SSH密钥（省去每次输入密码的步骤）

  • 继续在桌面 Git Bash, 输入：
  • 接着来上三个回车
  • 到 Git Bash 的地址（桌面），C:\Users\yourname.ssh，找到文件 id_rsa.pub，用记事本打开，复制全部内容
  • 回到 github 主页，选择 Settings
  • 在左方的 Personal settings 选择 SSH and GPG keys
  • 点击绿色按钮 New SSH key
  • 在 Key 中粘贴刚刚复制的内容，Title 随意
  • 回到 Git Bash 检测是否配置成功，输入
    • 出现 Hi,… 说明成功

四、安装 node.js

  • 到 nodejs.org下载安装
  • 完成后打开命令行（Windows+R+回车），输入
    • node -v
    • npm -v
    • 出现版本信息说明安装成功

五、Hexo(核心：博客网站的框架)

  • 在你需要写博客的地方创建一个文件夹，可以命名为 Blog
  • 打开命令行，进入刚刚创建的文件夹中，可以效仿下图，也可以在文件夹内 Git Bash
    • 输入 npm install -g hexo-cli
    • 完成后再输入 hexo init blog
  • 耐心等待一段时间，可以看点有趣的小电影
  • 出现下图所示信息就差不多了

六、博客在本地初步形成（localhost:4000）

  • 进入刚刚生成的子文件夹 hexo
  • 右键 Git Bash Here
    • 输入 hexo g
    • 再输入 hexo s
    • 出现如下信息后，到浏览器进入 localhost:4000 , 便可以看到你的博客框架
  • 到此基本操作已经完成了，感动否
  • 不过别激动，现在只是在本地预览，接下来需要推送到远端
  • 输入命令 hexo -help 可以查看关于 hexo 操作的相关帮助

七、博客的推送与部署

  • 首先重要的一点，先理清两个概念（不要搞混了）：
    • 根目录 blog 下的 _config.yml 文件称为站点配置文件
    • 主题目录 blog 下的 _config.yml 文件称为主题配置文件
  • 打开站点配置文件 _config.yml，到最下方做如下修改（注意中间有空格）
  • 保存站点配置文件
  • 在根目录blog下输入命令
    • npm install hexo-deployer-git –save
  • 再依次输入下面三条命令：
    • hexo clean
    • hexo g
    • hexo d
  • 恭喜，现在你即可通过 yourname.github.io 访问到你的博客了

八、写文章

  • 命令行转到根目录，在根目录 Git Bash 也行
  • 输入 hexo new “文章名”
  • 到目录 \Blog\blog\source_posts下你可以看到你新生成的md文档，编辑保存后三步部署
    • hexo clean
    • hexo g
    • hexo d
  • 过一会刷新网页即可看到你的文章
  • 即刻介绍完毕，其他优化后续会推出！
